# GymTracker — Дневник тренировок

GymTracker — это веб-приложение для отслеживания спортивных тренировок, созданное на Django. Проект позволяет вести журнал выполненных упражнений и количеством повторений. 

## Ключевые возможности

*   
*   **Управление тренировками:** Полный цикл операций (CRUD) для создания, просмотра, редактирования и удаления тренировок, упражнений и подходов.
*   **Аналитика прогресса:** Страница со статистикой и графиками, показывающими динамику изменения рабочих весов.
*   **Адаптивный интерфейс:** Удобный и современный интерфейс.



Следуйте этим инструкциям, чтобы запустить копию проекта на вашем локальном компьютере для разработки и тестирования.

### Предварительные требования

Перед началом убедитесь, что у вас установлены:
*   Python (версии 3.9 или выше)
*   Менеджер пакетов Pip
*   Git

### Установка

1.  **Клонируйте репозиторий**
    Откройте терминал и выполните команду, чтобы получить копию проекта:
    ```bash
    git clone https://github.com/daaangit/Python-project.git
    cd Python-project
    ```

2.  **Настройте виртуальное окружение (рекомендуется)**
    Создайте и активируйте виртуальное окружение для изоляции зависимостей проекта:
    ```bash
    # Для Windows
    python -m venv venv
    venv\Scripts\activate
    # Для Linux или macOS
    python -m venv venv
    source venv/bin/activate
    ```

3.  **Установите зависимости**
    Установите все необходимые для работы проекта библиотеки, перечисленные в файле `requirements.txt`:
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте базу данных**
    Примените миграции Django для создания структуры базы данных:
    ```bash
    python manage.py migrate
    ```

5.  **Создайте учетную запись администратора (опционально)**
    Чтобы получить доступ к встроенной административной панели Django, создайте суперпользователя:
    ```bash
    python manage.py createsuperuser
    ```
    Следуйте инструкциям в терминале, чтобы задать имя пользователя и пароль.

6.  **Запустите сервер**
    Запустите встроенный сервер для разработки:
    ```bash
    python manage.py runserver
    ```

7.  **Откройте приложение**
    Перейдите в браузере по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000). Вы должны увидеть главную страницу приложения.
